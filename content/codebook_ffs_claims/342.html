ICD_PRCDR_VRSN_CD25
LABEL:
Claim Procedure Code XXV Claim Procedure Version Code (ICD-9 or ICD-10)
DESCRIPTION:
The code used to indicate if the procedure code is ICD-9 or ICD-10.
SHORT NAME:
ICD_PRCDR_VRSN_CD25
LONG NAME:
ICD_PRCDR_VRSN_CD25
TYPE:
CHAR
LENGTH:
1
SOURCE:
NCH
VALUES:
Blank = ICD-9
9 = ICD-9
0 = ICD-10
COMMENT:
On October 1, 2015 the conversion from the 9th version of the International
Classification of Diseases (ICD-9-CM) to version 10 (ICD-10-PCS) occurred.

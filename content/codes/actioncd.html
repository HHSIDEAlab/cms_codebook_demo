ACTIONCD
LABEL:
FI or MAC Claim Action Code
DESCRIPTION:
The type of action requested by the intermediary to be taken on an institutional claim.

ACTIONCD
LONG NAME:
FI_CLM_ACTN_CD
TYPE:
CHAR
LENGTH:
1
SOURCE:
NCH
VALUES:
1 =
Original debit action (always a 1 for all regular bills)
5 =
Force action code 3 (secondary debit adjustment)
8 =
Benefits refused

CLM_CARE_IMPRVMT_MODEL_CD4
LABEL:
Claim Care Improvement Model 4 Code
DESCRIPTION:
This code is used to identify that the care improvement model 4 is being used for
payments.
SHORT NAME:
CLM_CARE_IMPRVMT_MODEL_CD4
LONG NAME:
CLM_CARE_IMPRVMT_MODEL_CD4
TYPE:
CHAR
LENGTH:
2
SOURCE:
NCH
VALUES:
64 = Care Improvement Model 4 is used
Null/missing
COMMENT:
There are 4 of these Care Improvement Model fields
(CLM_CARE_IMPRVMT_MODEL_CD1-CLM_CARE_IMPRVMT_MODEL_CD4).
This field was new in 2013, and is null/missing for all previous years.

CLM_CARE_IMPRVMT_MODEL_CD1
LABEL:
Claim Care Improvement Model 1 Code (bundled payment)
DESCRIPTION:
This code is used to identify that the care improvement model 1 is being used for
bundling payments. The initiative if referred to as the Bundled Payments for Care
Improvement initiative (BPCI, Model 1).
SHORT NAME:
CLM_CARE_IMPRVMT_MODEL_CD1
LONG NAME:
CLM_CARE_IMPRVMT_MODEL_CD1
TYPE:
CHAR
LENGTH:
2
SOURCE:
NCH
VALUES:
61 = Care Improvement Model 1 is used
Null/missing
COMMENT:
There are 4 of these Care Improvement Model fields
(CLM_CARE_IMPRVMT_MODEL_CD1-CLM_CARE_IMPRVMT_MODEL_CD4).
This field was new in 2013, and is null/missing for all previous years.

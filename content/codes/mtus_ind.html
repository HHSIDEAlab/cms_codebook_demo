MTUS_IND
LABEL:
Carrier Line Miles/Time/Units/Services (MTUS) Indicator Code
DESCRIPTION:
Code indicating the units associated with services needing unit reporting on the line
item for the carrier claim (non-DMERC).
SHORT NAME:
MTUS_IND
LONG NAME:
CARR_LINE_MTUS_CD
TYPE:
CHAR
LENGTH:
1
SOURCE:
NCH
VALUES:
0 =
Values reported as zero (no allowed activities)
1 =
Transportation (ambulance) miles
2 =
Anesthesia time units
3 =
Services
4 =
Oxygen units
5 =
Units of blood

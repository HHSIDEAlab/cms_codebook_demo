CLM_VBP_PRTCPNT_IND_CD
LABEL:
Claim Value-Based Purchasing (VBP) Participant Indicator Code
DESCRIPTION:
This field is the code used to identify a reason a hospital is excluded from the Hospital
Value Based Purchasing (HVBP) program.
SHORT NAME:
CLM_VBP_PRTCPNT_IND_CD
LONG NAME:
CLM_VBP_PRTCPNT_IND_CD
TYPE:
CHAR
LENGTH:
1
SOURCE:
NCH
VALUES:
Y = Participating in Hospital Value Based Purchasing
N = Not participating in Hospital Value Based Purchasing
Null/missing = same as 'N'
COMMENT:
The ACA (Section 3001) excludes from the HVBP hospitals that meet certain conditions.
Additional information is available on the CMS "Hospital Value-Based Purchasing"
website.
This initiative began in 4th Quarter of 2013 (i.e., beginning of Federal fiscal year 14).
This field was new in 2013, and is null/missing for all previous years.

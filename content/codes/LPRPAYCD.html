LPRPAYCD
LABEL:
Line Primary Payer Code (if not Medicare)
DESCRIPTION:
The code specifying a federal non-Medicare program or other source that has primary
responsibility for the payment of the Medicare beneficiary's medical bills relating to the
line item service on the non-institutional claim.
The presence of a primary payer code indicates that some other payer besides Medicare
covered at least some portion of the charges.

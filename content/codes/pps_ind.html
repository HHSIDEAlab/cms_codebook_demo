PPS_IND
LABEL:
Claim PPS Indicator Code
DESCRIPTION:
The code indicating whether or not:
(1) the claim is from the prospective payment system (PPS), and/or
(2) the beneficiary is a deemed insured MQGE (Medicare Qualified Government
Employee)

PPS_IND
LONG NAME:
CLM_PPS_IND_CD
TYPE:
CHAR
LENGTH:
1
SOURCE:
NCH

Blank
=
Not a PPS bill
2
=
PPS bill; claim contains PPS indicator

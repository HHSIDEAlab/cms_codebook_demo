PROVIDER
LABEL:
Provider Number
DESCRIPTION:
This variable is the provider identification number.
The first two digits indicate the state where the provider is located, using the SSA state
codes; the middle two characters indicate the type of provider; and the last two digits
are used as a counter for the number of providers within that state and type of provider
(i.e., this is a unique but not necessarily sequential number).

PROVIDER

CARETHRU
LABEL:
NCH Active or Covered Level Care Thru Date
DESCRIPTION:
The date on a claim for which the covered level of care ended in a general hospital or
the active care ended in a psychiatric/tuberculosis hospital.
SHORT NAME:
CARETHRU
LONG NAME:
NCH_ACTV_OR_CVRD_LVL_CARE_THRU
TYPE:
DATE
LENGTH:
8
SOURCE:
NCH QA Process
VALUES:
-
COMMENT:
This variable is derived, using the occurrence code (variable called
CLM_RLT_OCRNC_CD), when the value is 22.
When this code value is present the date
is populated using the CLM_RLT_OCRNC_DT.

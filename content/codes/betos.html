Line Berenson-Eggers Type of Service (BETOS) Code
DESCRIPTION:
The Berenson-Eggers type of service (BETOS) for the procedure code based on generally
agreed upon clinically meaningful groupings of procedures and services.
This field is included as a line item on the non-institutional claim.

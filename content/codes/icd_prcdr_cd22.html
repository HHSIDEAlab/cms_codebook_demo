ICD_PRCDR_CD22
LABEL:
Claim Procedure Code XXII
DESCRIPTION:
The code that indicates the procedure performed during the period covered by the
institutional claim.
SHORT NAME:
ICD_PRCDR_CD22
LONG NAME:
ICD_PRCDR_CD22
TYPE:
CHAR
LENGTH:
7
SOURCE:
NCH
VALUES:
-
COMMENT:
Effective July 2004, ICD-9-CM procedure codes are no longer being accepted on
Outpatient claims.
The ICD-9-CM codes were named as the HIPPA standard code set for inpatient hospital
procedures.
HCPCS/CPT codes were named as the standard code set for physician services and other
health care services.
